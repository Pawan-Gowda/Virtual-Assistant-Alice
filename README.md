# Overview

Alice is a Python-based virtual assistant that interacts via voice or text input.
It can perform tasks like opening apps, playing music, fetching news, checking weather, reading PDFs, performing calculations, and more.
Built using Python libraries, APIs, and a simple GUI, Alice is beginner-friendly and expandable.
